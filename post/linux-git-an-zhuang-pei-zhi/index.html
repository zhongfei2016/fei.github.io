linux git安装配置

2019-07-29

安装

直接在root用户下执行：

[root@fei ~]# cd /home/fei/tools/
[root@fei tools]# yum install git

配置

同样root用户下执行：

[root@fei codes]# git config --global user.name "zhongfei2016"
[root@fei codes]# git config --global user.email zhongfei1217@163.com

然后生成公钥和私钥，直接一路回车即可：

[root@fei .ssh]# ssh-keygen -t rsa -C "zhongfei1217@163.com"

默认生成的公钥和私钥都在/root/.ssh目录下，可以更改下权限，避免后续出现权限问题：

[root@fei .ssh]# ll
total 8
-rw------- 1 root root 1675 Jul 29 23:39 id_rsa
-rw-r--r-- 1 root root  402 Jul 29 23:39 id_rsa.pub
[root@fei .ssh]# pwd
/root/.ssh
[root@fei .ssh]# chmod 700 id_rsa
