flink安装调测说明

2019-07-28

下载

下面是下载地址，从镜像地址下载会比较快，尤其是国内的教育站点： https://flink.apache.org/downloads.html

安装启动

将tgz包放到Mac/Linux设备目录上，我这里有一个旧笔记本安装的CentOS 6.9，所以就直接放在上面的/home/fei/tools目录下了，解压： 然后直接进到bin目录下执行启动脚本即可： 此时可以直接访问http://192.168.0.107:8081，页面会跳转到flink自带的portal页面，可以看到当前有几个可用的slot以及使用情况。

测试程序

可以到example目录下执行一下测试程序，执行前首先要起一个服务端端口用来监听数据流，模拟不断输入的数据端，可以使用netcat来起端口： nc -l -p 9990 起来后可以使用netstat来查看端口是否有在监听： 如果启动报错： lnvalid maximum direct memory size: -XX: MaxDirectMemorySize=8388607T

可以将/home/fei/tools/flink-1.7.2/bin目录下的taskmanager.sh的TM_MAX_OFFHEAP_SIZE改为"1024M"，默认值是"8388607T"，可以注释备份并保存：
