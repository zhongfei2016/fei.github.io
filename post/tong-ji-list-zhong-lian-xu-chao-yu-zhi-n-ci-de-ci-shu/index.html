统计list中连续超阈值n次的次数

2019-09-13

一个Double类型的list，求取此list中元素值连续大于一个阈值n次（n可以修改）的总次数，比如阈值是4，list中所有元素为：1,5,7,3,4,8,11,1,7,1，要求连续2次超过阈值才计次，那么结果应该是2次，5,7一次，4,8,11一次，7虽然大于4，但由于前后均小于4，所以不计次，代码实现如下：

    /**
     * 
     * @param list 要统计的列表
     * @param threshold 阈值
     * @param count 要求超阈值连续次数
     * @return
     */
    public static int calExceedNum(List<Double> list, double threshold, int count)
    {
        //符合要求的次数
        int cnt = 0;
        //单次连续超阈值的次数
        int continueCnt = 0;
        for (int i = 0; i < list.size(); i++)
        {
            if (list.get(i) < threshold)
            {
                continueCnt = 0;
                continue;
            }
            else
            {
                continueCnt++;
            }
            if(continueCnt==count)
            {
                cnt++;
            }
        }
        return cnt;
    }
